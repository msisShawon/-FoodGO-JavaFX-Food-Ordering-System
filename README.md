## рџљЂ Getting Started

### Prerequisites

- Java JDK 17 or later
- JavaFX SDK
- IntelliJ IDEA
- Scene Builder

### Run the Project

1. Clone the repository

```bash
git clone https://github.com/your-username/foodgo-javafx-food-ordering-system.git
```

2. Open the project in IntelliJ IDEA.

3. Configure the JavaFX SDK.

4. Add the required VM options:

```text
--module-path PATH_TO_FX/lib --add-modules javafx.controls,javafx.fxml
```

5. Run `Main.java`.
